# PageNow（基于VUE的数据可视化开发平台）

### 介绍
基于Vue+iView的可视化页面开发平台，拖拽式设计，独创布局块概念，基于iView的栅格系统原生支持响应式栅格布局，以获得更好的移动设备兼容性，集成百度Echarts以实现丰富的数据可视化配置，原生通用组件满足你各种可视化页面的开发需求。
